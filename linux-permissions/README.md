# Linux File Permissions Management

## Objective
To analyze and manage Linux file and directory permissions to reduce unauthorized access and improve system security.

## Environment
- Operating System: Linux (Ubuntu)
- Tools: Terminal, chmod, chown, ls

## Process
- Reviewed existing file and directory permissions
- Modified permissions using chmod
- Verified ownership using chown
- Validated changes with ls -l

## Findings
Incorrect permissions can expose sensitive files. Proper permission management significantly reduces attack surface.

## Skills Demonstrated
- Linux access control
- Command-line proficiency
- Security hardening principles
